# Prode-2026
Prode interactivo para el mundial 2026, pensado para un grupo chico de familia y amigos.

## Importante

Esta app es un cliente estático que guarda datos en Supabase desde el navegador. Los PINes y la clave de admin ayudan a evitar cambios accidentales, pero no son seguridad fuerte: alguien con conocimiento técnico puede inspeccionar el código y las llamadas.

Usala para un grupo de confianza. Si se quiere abrir al público o usar con premios importantes, conviene mover la autenticación y las escrituras sensibles a un backend o a reglas de Supabase por usuario.
